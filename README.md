# unive-imdb-api

## How to run the API

- Install the dependencies.

```bash
npm i
```

- Compile the Typescript files.

```bash
tsc # Make sure to have Typescript installed globally.
```

- Run the Express server via _npm command._

### Profiling mode (dev)

```bash
npm run dev-profile [filename]
```

### Profiling mode (prod)

```bash
npm run profile [filename]
```

### Testing mode (dev)

```bash
npm run dev-test
```

### Testing mode (prod)

```bash
npm run test
```
